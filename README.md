# Clinkd

Clinkd is a simple web app built using React to improve communication and coordination among students on campus.

## Features

- Event creation and sharing
- Announcement posts
- Group-based communication

## Tech Stack

- React
- JavaScript
- CSS

## How to Run

```bash
git clone https://github.com/Mystic-commits/Clinkd.git
cd Clinkd
npm install
npm start
